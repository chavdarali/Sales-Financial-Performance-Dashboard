# Sales-Financial-Performance-Dashboard
Analyse sales call centre performance and identify operational and financial KPIs using Excel, SQL and Power BI.

Tools(Excel, SQL, SQLite, Power BI, DAX)

Skills (Data Cleaning, Financial Analysis, KPI Reporting, SQL Queries, Dashboard Design, Business Intelligence, Data Visualisation)

KPIs (Total Calls, Products Sold, Revenue, Pickup Rate, Conversion Rate, Average Call Duration, Profit)

Business Insights:

Gloria Singh generated the highest sales volume.
The overall pickup rate was close to 70%.
Conversion performance varied by agent.
Average call duration was approximately 125 seconds.
Agent-level performance differences highlight opportunities for coaching and process improvement.
